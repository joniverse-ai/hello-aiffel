# 안녕하세요, 박정목입니다 👋

## 이름
박정목

## 내 도메인
나는 상품이 하고싶어 하는 말을 소비자에게 디자인으로 쉽게 전달하는 사람이다.

## 이 과정에서 만들고 싶은 것 3가지
> 브랜드·상품 디자인 작업의 반복되는 흐름(리서치 → 핵심 메시지 도출 → 비주얼 전환 → 가이드라인 문서화)을 AI로 자동화하는 것을 목표로 합니다.

1. **상품 메시지 추출 도구** — 상품 정보·기획서를 입력하면 "상품이 소비자에게 하고 싶은 핵심 메시지"를 자동으로 정리해주는 에이전트
2. **메시지 → 비주얼 변환 시스템** — 도출된 메시지를 패키지·상세페이지·키비주얼 등 소비자가 쉽게 이해할 디자인 안으로 전환하는 워크플로
3. **리서치·가이드라인 자동화** — 경쟁사/레퍼런스 리서치와 디자인 규칙 문서화·검수 과정을 자동화하는 도구
